## Overview

Perplexity Advanced is an AI-powered search and research application that answers questions with sourced, cited responses drawn from the web, providing the depth of AI-generated answers with the verifiability of traditional search through inline citations that link to the original sources. It reimagines web search as a conversational research tool where questions receive comprehensive answers rather than a list of links, while maintaining the transparency of source attribution that factual research requires.

AI-powered answers synthesize information from multiple web sources into coherent, comprehensive responses that directly answer the question rather than requiring the user to read multiple web pages and synthesize the answer themselves. Each claim in the response includes citations linking to the source material, enabling verification of any specific point. The answers handle factual questions, comparisons, explanations, and multi-part research queries.

Follow-up questions enable iterative research where each answer generates suggested follow-up queries that deepen the investigation. The conversational flow enables exploring a topic progressively, with each question building on the context of previous answers. This iterative approach handles the common research pattern of starting with a broad question and progressively narrowing to specific details.

Focus modes direct the AI to search specific source types — academic papers, YouTube videos, Reddit discussions, news, or the general web — providing control over the type of sources that inform the answer. Academic focus draws from scholarly publications for research questions, while Reddit focus captures community experience and opinions.

For researchers, students, professionals, and anyone who needs well-sourced answers to questions rather than a list of links to evaluate, Perplexity Advanced is the research tool that combines AI synthesis with source transparency.

## Performance

Perplexity balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
